# Assets Images

Folder ini berisi gambar-gambar yang digunakan dalam aplikasi.

## Gambar yang Diperlukan:

1. **ranca_upas.jpg** - Gambar tempat wisata Ranca Upas
2. **kawah_putih.jpg** - Gambar tempat wisata Kawah Putih
3. **tangkuban_perahu.jpg** - Gambar tempat wisata Tangkuban Perahu
4. **ulbi.jpg** - Gambar logo ULBI (untuk praktikum image asset)

## Cara Menambahkan Gambar:

1. Download atau siapkan gambar dengan nama sesuai di atas
2. Letakkan file gambar di folder `assets/images/`
3. Pastikan nama file sesuai dengan yang didefinisikan di `tourism_place.dart`

## Sumber Gambar:

Anda bisa mendapatkan gambar dari:
- Google Images (pastikan free to use)
- Unsplash.com
- Pexels.com
- Atau foto pribadi

## Format yang Disarankan:

- Format: JPG atau PNG
- Resolusi: Minimal 800x600 pixels
- Ukuran file: Maksimal 500KB per gambar
